# MSME AI Platform

An AI-powered platform designed to empower Micro, Small, and Medium Enterprises (MSMEs) by providing intelligent assistance for business growth, government scheme discovery, financial insights, document analysis, and workflow automation.

## Features

- 🤖 AI Business Assistant
- 📑 Government Scheme Discovery
- 💰 Loan & Subsidy Eligibility Checker
- 📄 Document Analysis (GST, UDYAM, PAN, etc.)
- 📊 Business Analytics Dashboard
- 📈 Sales & Financial Insights
- 🧾 Invoice & Expense Management
- 💬 Multilingual AI Chatbot
- 📅 Compliance & Tax Reminders
- 📦 Inventory & Order Management
- 📱 WhatsApp Integration
- 🔔 Smart Notifications

## AI Capabilities

- Natural Language Question Answering
- Personalized Business Recommendations
- Scheme Eligibility Prediction
- Intelligent Document Processing
- OCR for Business Documents
- Financial Trend Analysis
- Risk Assessment
- AI-powered Report Generation

## Installation

```bash
git clone https://github.com/your-username/msme-ai-platform.git

cd msme-ai-platform

# Install frontend
cd frontend
npm install

# Install backend
cd ../backend
pip install -r requirements.txt
```

## Running the Project

Frontend

```bash
npm run dev
```

Backend

```bash
uvicorn main:app --reload
```
